# Project Brief: Blood Bowl Card Creator

## Overview
A web-based custom card creator for Blood Bowl tabletop game players. The tool allows users to create, customize, and export professional-looking player cards, special cards, and playbook cards for their Blood Bowl teams.

## Core Purpose
To provide Blood Bowl players with an easy-to-use tool for creating custom player cards that match the official game aesthetic while supporting homebrew content and customization.

## Key Features

### Player Cards
- Create custom player cards with complete stats (MA, ST, AG, PA, AV)
- Support for both roster players and star players
- Skill category selection (Primary and Secondary)
- Custom player images with positioning controls
- Skills & traits text area
- Position and team name customization
- Gold piece (GP) cost display

### Skill Categories (2025 Rules)
The application supports the full 2025 Blood Bowl skill category system:
- **A** — Agility
- **D** — Devious
- **G** — General
- **M** — Mutation
- **P** — Passing
- **S** — Strength

### Special Cards
- Custom special rules cards
- Flexible text layout with timing, duration, and effect sections
- Support for special plays and abilities

### Playbook Cards
- Team playbook strategy cards
- Custom objectives and actions
- Visual organization for game strategies

## User Experience Goals
1. **Simple & Intuitive**: No learning curve, start creating immediately
2. **Professional Output**: Cards look official and print-ready
3. **Flexible**: Support both official and homebrew content
4. **Persistent**: Save and load card designs via JSON export/import
5. **Browser-Based**: No installation required, works anywhere

## Technical Constraints
- Must work in modern web browsers
- Fully client-side (no backend required)
- Export to PNG for printing
- Export to JSON for saving/sharing designs
- Maintain visual consistency with Blood Bowl aesthetic

## Success Metrics
- Cards render correctly with all stat combinations
- Export functionality works reliably
- Design persists across browser sessions
- Users can easily create cards within minutes

## Future Considerations
- Potential for team roster management
- Card back designs
- Additional card types (inducements, referee cards)
- Multi-language support
